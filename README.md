# Database Setup

1. Open SQL Server Object Explorer in Visual Studio.
2. Expand (localdb)\MSSQLLocalDB.
3. Right-click Databases → Add New Database.
4. Name it: BookCollectionDB
5. Right-click the new database → New Query.
6. Copy/paste everything from DatabaseScripts/BookCollectionDB.sql.
7. Run the script.
